# Images Directory

Place your images here:

## Required Images:

### Profile & Headers
- `avatar.jpg` - Your profile photo (square, 400x400px recommended)
- `header-piano.jpg` - Homepage header (1920x600px recommended)
- `music-header.jpg` - Music page header (1920x600px recommended)
- `writing-header.jpg` - Writing page header (1920x600px recommended)

### Album Art
- `water-memories-cover.jpg` - Water Memories album art (square, 3000x3000px for best quality)
- `monthly-series.jpg` - Generic monthly series image
- Additional album art for each release (square format)

## Image Guidelines:

- Use high-quality images
- Keep file sizes reasonable (compress for web)
- Use consistent aspect ratios for headers (1920x600)
- Album art should be square (1:1 ratio)
- Recommended formats: JPG for photos, PNG for graphics with transparency

## Optimization Tips:

You can compress images before uploading using:
- TinyPNG (https://tinypng.com/)
- Squoosh (https://squoosh.app/)
- ImageOptim (Mac)

This will help your site load faster without sacrificing quality.
